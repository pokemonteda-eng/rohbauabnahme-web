# Frontend

Frontend-Basis für `rohbauabnahme-web` mit:

- React 18 + TypeScript
- Vite 5
- Tailwind CSS (inkl. PostCSS-Konfiguration)
- shadcn/ui Basis-Konfiguration (Alias, `components.json`, `cn()` Helper, erste UI-Komponente)

## Lokaler Start

```bash
npm install
npm run dev
```

## Build-Check

```bash
npm run lint
npm run build
```
